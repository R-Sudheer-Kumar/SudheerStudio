# SudheerStudio Website

A premium, single-page multi-view website for SudheerStudio - an Indian indie developer studio.

## 🚀 Features
- **Obsidian Flux Design**: High-performance "Editorial" aesthetic geared towards software engineers.
- **SPA Routing**: Instant page switching using modern CSS-animated views and Hash-based routing.
- **Bharat Optimized**: Localization features for Indian payment systems and contact channels.
- **Premium Assets**: AI-generated hero and product mockups.

## 🎨 Design System: Obsidian Flux
The website utilizes the **Obsidian Flux** design system, a high-end editorial framework:
- **Tonal Depth**: Layered backgrounds (from #0e0e0e to #131313) instead of borders.
- **Typography**: Inter (Body) + JetBrains Mono (Utility).
- **Accents**: Electric Blue (#006ff0) and Amber Gold (#f59e0b).
- **Glassmorphism**: Subtle ambient glows and frosted overlays.

## 📂 Structure
- `index.html`: The core application containing CSS, HTML, and JS.
- `assets/`: Custom product imagery and branding assets.

## 🛠 Tech Stack
- Vanilla HTML5 / CSS3
- Modern ES6+ JavaScript
- Google Fonts (Inter, JetBrains Mono)
